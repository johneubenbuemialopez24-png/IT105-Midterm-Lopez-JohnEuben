# Appointment System Database

## Description
This project is a simple Appointment System database that manages patients, doctors, appointments, and payments.

## Tables
- Patients: Stores patient information
- Doctors: Stores doctor details
- Appointments: Stores appointment schedules
- Payments: Stores payment records

## Features
- CRUD operations
- JOIN queries
- Subqueries
- Normalization (UNF–3NF)
- Indexing

## Reflection
I learned how to design a database, normalize data, write SQL queries, and improve performance using indexing. I also learned how to use GitHub for version control.
